# Lucas's dotfiles

only dependency (besides the programs themselves) is bash or zsh

Clone into .dotfiles and install:
```bash
git clone https://github.com/ikzLucas/lk.dotfiles.git ~/.dotfiles && cd ~/.dotfiles && chmod +x install.sh && ./install.sh
```
